Что такое СSR

Программа CSR - представляет собой простую в освоении, но мощную программу для работы с данными и услугами клиентов.

Структура проекта

Для построения программы использовали патерн MVC, Maven для сборки, JavaFX для View и GSON для работы с JSON файлами.
